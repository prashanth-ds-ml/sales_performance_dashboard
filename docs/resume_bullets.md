# Resume Bullets

- Built a SQL-first sales analytics dashboard in Streamlit to monitor revenue, profit, orders, discounts, regional performance, product rankings, and loss-making segments using reusable SQLite queries.
- Implemented filter-aware SQL using a CTE wrapper so date, segment, region, and category filters apply consistently across saved analysis queries.
- Created a reproducible CSV-to-SQLite pipeline with normalized columns, typed date/numeric fields, and indexed dimensions for dashboard filtering.
